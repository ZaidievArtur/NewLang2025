 Тема: Конвертация списка объектов в простую JSON-строку 

 Цель работы: Реализовать алгоритм конвертации списка объектов (Case Classes) в формат JSON-строки на языке Scala с использованием библиотеки Jackson. В ходе работы необходимо настроить взаимодействие между типобезопасными структурами Scala и движком сериализации через модуль DefaultScalaModule, применить механизмы интроспекции объектов для автоматического извлечения данных и обеспечить читаемость выходного потока с помощью форматирования (Pretty Printing).
