Тема: Преобразование типа данных в массиве 

Цель работы: Реализовать эффективное преобразование типов данных в массивах Julia. В ходе работы необходимо применить точечную нотацию для векторизации вычислений и интегрировать методы математического округления для обеспечения корректного приведения типов без потери точности.
