Тема: Функция поиска наибольшего общего делителя (НОД) 

Цель работы: Реализовать алгоритм поиска наибольшего общего делителя (НОД) с использованием классического метода Евклида. В ходе работы необходимо применить механизм рекурсии для итеративного вычисления остатка, использовать сопоставление с образцом (Pattern Matching) для обработки базового случая и обеспечить хвостовую оптимизацию вызовов для эффективного использования памяти.
