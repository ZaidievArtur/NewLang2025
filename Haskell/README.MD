Тема: Реализация Quicksort 

Цель работы: Решить задачу эффективной сортировки данных (на примере алгоритма Quicksort). В ходе работы необходимо реализовать рекурсивную декомпозицию списка, применить механизм сопоставления с образцом (Pattern Matching) и генераторы списков (List Comprehensions) для лаконичного описания логики разделения данных.
