Тема: Асинхронный симулятор загрузки данных 

Цель работы: Решить задачу параллельной обработки данных (симуляция загрузки нескольких ресурсов), используя механизмы структурированного параллелизма Swift. При этом необходимо обеспечить отзывчивость программы через асинхронное ожидание (Async/Await) и реализовать устойчивость к сбоям через изолированную обработку ошибок в многопоточной среде.
